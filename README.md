# Staff & Ledger Website

Astro and TinaCMS marketing site for Staff & Ledger.

## Local Development

```sh
npm install
npm run dev
```

To run with TinaCMS locally:

```sh
npm run tina
```

## Build

```sh
npm run build
```

The production host should run `tinacms build` before `astro build` so `/admin` is generated.
